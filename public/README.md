# Warhammer Character Builder Assets

This directory contains assets needed for the Warhammer Character Builder interface.

## Required Images

Place the following images in the `/public/images/` directory:

- `warhammer-logo.webp` - Main logo for splash screen (dark Warhammer logo)
- `warhammer-banner.webp` - Banner graphic for the top of the page
- `warhammer-emblem.webp` - Small emblem for the stats section
- `warhammer-secondary-icon.webp` - Icon for the secondary attributes section
- `warhammer-corner-decor.webp` - Decorative element for bottom left corner
- `warhammer-corner-decor-2.webp` - Decorative element for top right corner
- `warhammer-stats-icon.webp` - Background watermark for stats tab
- `warhammer-skills-icon.webp` - Background watermark for skills tab
- `warhammer-talents-icon.webp` - Background watermark for talents tab
- `warhammer-combat-icon.webp` - Background watermark for combat tab
- `warhammer-equipment-icon.webp` - Background watermark for equipment tab
- `warhammer-info-icon.webp` - Background watermark for info tab
- `gold-coin.webp` - Gold coin icon (6x6px)
- `silver-coin.webp` - Silver coin icon (6x6px)
- `copper-coin.webp` - Copper coin icon (6x6px)

## Required Textures

Place the following textures in the `/public/textures/` directory:

- `dark-texture.webp` - Main background texture (dark, gritty)
- `parchment.webp` - Aged parchment texture for stats tab
- `leather.webp` - Leather texture for skills tab
- `magic-scroll.webp` - Magical scroll texture for talents tab
- `armory.webp` - Metal/armor texture for combat tab
- `leather-pouch.webp` - Leather bag texture for equipment tab
- `old-map.webp` - Ancient map texture for info tab

## Style Guide

- Primary colors: 
  - Deep reds: `#540804`, `#AA3C3B`, `#380303`
  - Rich browns: `#2A1A17`, `#1D1210`, `#513428`
  - Accent gold: `#C09E6B`, `#E4D8B4`
  - Text colors: `#E4D8B4` (light text), `#867E70` (muted text)

- Typography:
  - Serif fonts for headers and main text
  - Dark fantasy aesthetic

## Asset Guidelines

- Use high-quality images with a dark, gritty Warhammer fantasy aesthetic
- Ensure all images have transparent backgrounds where appropriate
- Keep file sizes optimized for web (use WebP format)
- Maintain a consistent visual style across all assets
